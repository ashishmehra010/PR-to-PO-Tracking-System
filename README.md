# PR to PO Procurement Tracking System

## Project Overview
This project simulates an end-to-end procurement workflow from Purchase Requisition (PR) creation to Purchase Order (PO) completion.

## Objectives
- Track PR and PO status
- Measure procurement cycle time
- Improve process transparency

## Tools Used
- Microsoft Excel
- Data Validation
- Date Calculations

## Workflow Covered
- PR Creation
- Approval Process
- PO Issuance
- Delivery & Payment Tracking

## Key Metrics
- PR to PO Cycle Time
- Pending Approvals
- Delayed Orders

## Outcome
- Improved procurement process visibility
- Reduced delays in approvals
- Better reporting accuracy

## Resume Highlight
Designed a PR-to-PO tracking system to monitor procurement cycle times and approval status.
